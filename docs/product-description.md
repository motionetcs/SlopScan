# SlopScan Product Description

## One-Line Description

SlopScan is a privacy-conscious review trust analyzer that helps shoppers see the review patterns behind a product rating before they buy.

## Short Description

SlopScan lets users paste a product URL, paste or import review text, and generate an explainable trust dashboard. It flags repeated praise, generic promotional wording, product mismatch, unsupported seller claims, weak Q&A answers, and real buyer issue clusters.

## Presentation Description

SlopScan is built for real online shopping decisions. Star ratings can hide repetitive five-star praise, paid-style language, low-detail reviews, or reviews that do not even match the product. SlopScan gives shoppers a second opinion by analyzing the review text they already see on marketplaces.

The flow is simple: paste a product URL, paste or import reviews, click Analyze, and read the report. The dashboard shows a trust score, review-risk score, suspicious review share, grounded review percentage, buyer issue map, Review X-Ray, seller-claim support checks, and a copyable report summary.

SlopScan does not claim to prove fake reviews. It surfaces explainable signals so shoppers can inspect the evidence and make better decisions.

## Demo Script

I start by pasting a product URL. SlopScan detects the marketplace and moves me into the review analysis flow. Since marketplaces often restrict direct review access, the app asks me to paste or import the exact reviews I want checked.

After I paste reviews and click Analyze Reviews, SlopScan creates a dashboard with the trust score, review-risk score, repeated phrase clusters, grounded buyer issues, suspicious review explanations, and seller-claim support. This helps me understand whether the rating is supported by useful buyer evidence or inflated by vague review noise.
